## Data Flow

```
Source Data → CLEAN Layer → STAGE Layer → MODEL Layer → GOLD Layer → Analysis
```

## Layers

### CLEAN Layer
- Schema: `FIBER_MAP.CLEAN`
- Raw data, no transformations
- Source of truth

### STAGE Layer
- Schema: `FIBER_MAP.DBT_STAGE`
- Standardized column names (UPPERCASE_UNDERSCORED)
- Appropriate data types
- Tables prefixed with `stg_`

### MODEL Layer
- Schema: `FIBER_MAP.DBT_MODEL`
- Normalized by business entity
- Derived analytical fields
- Tables prefixed with `mdl_`

### GOLD Layer
- Schema: `FIBER_MAP.DBT_GOLD`
- Denormalized for analysis
- Use case specific
- No prefix (e.g., `usa_all_periods_final`)
